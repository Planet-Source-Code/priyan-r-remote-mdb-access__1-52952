Properties(6)          

1. Dbpassword
   If your access file is password protected set this poperty
2. Fieldcount
  Returns the number of fields in query result
                      eg: If you execute this query select name,phone from tablename
                      The field count will be 2
3. Mdbfile
  The path of the MS-Access File on ther server
        eg: if your file is at yourserver/database/file.mdb and the server script is at
yourver/sript.asp then set database/file.mdb
4 Recordcount
  The count of the records returned by your sql query
5. Recordstofetch
  The number of records to download from the access file by default it is 100
6. Scripturl
  The server script url.I provided two sctipts asp and asp.net you can put one or both in the webserver

Methods/Functions(7)   

1. Executequery
  You need to to call this method to access all other mehods
   syntax: executequery(byval sql as string,optional pos as long)
           sql: the sql query to execute
           pos: the start position of record to download
           (optional bydefault executequery returns records from the record position 1) eg: if your sql query returns 4 records and you set pos to 2 then it gives the records from 2 to 4
2. Addnew
  You can use this to add new record. Syntax: Addnew(byval values as string)
      you need pass the values to add.Seperate fileds by |~|
     eg:addnew("field1=value|~|field2=value|~|field3=value)
3. Update
  you can use this to update a  record. Syntax: update(ByVal pos as long, values as string)
      pass value pos to the according position of the record to update
      you need pass the values to add.Seperate fileds by |~|
     eg:addnew("field1=value|~|field2=value|~|field3=value)
4. Delete
  Use this function to delete a record synatx: delete(ByVal pos as long)
     pass value pos to the according position of the record to Delete
5. Getfield
  Use this function to get the field name syntax:getfield(byval index as long)
           returns the fieldname use field count prperty to count the available fields
          eg: select name,phone from tablename
                       here field( 1) will return will give the field name 'name' and field(2) will give 'phone '
6. Getrow
  use this function to read a row of record.
          Syntax:getrow()
          Each field will be seperated by a |$|
         value1|$|value2|$|value3
6. Cancel
  Cancels the current execution

Events(1)

6. Status(ByVal status As String)
  Status  gives the current status description
